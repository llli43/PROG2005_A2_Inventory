Inventory Management System

Welcome to the Harvey Norman style inventory manager.

This application helps you manage product stock, suppliers, pricing, and popularity status.

Features

  • Add new items (unique ID required)
  • Update items by name
  • Delete items with confirmation
  • Search items by name
  • View only popular items
  • Display all items in a responsive table

Mobile Responsive

Designed to work perfectly on both narrow (mobile) and wide (desktop) screens.

Privacy & Security

All data stays in your browser session. No server-side storage.

See the Privacy page for details.
