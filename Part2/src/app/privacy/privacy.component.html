Privacy & Security

Data Storage

All inventory data is stored only in your browser's memory during the current session.

No data is sent to any server or persisted after you close the browser tab.

Security Measures

  • Session-only data storage
  • No external API calls
  • No tracking or analytics
  • Client-side only application

Best Practices

For production use, consider implementing:

  • Backend database with authentication
  • Input validation and sanitization
  • HTTPS encryption
  • Regular data backups
